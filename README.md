# Clouds & Bones
Volumtrisches Rendering von wissenschaftlichen Wetterdaten in Unity.
Das Projekt wurde unter der Leitung von Alexander Wollert entwickelt.

## Requirements
Das Projekt basiert auf Unity 2019.1.1f1.
